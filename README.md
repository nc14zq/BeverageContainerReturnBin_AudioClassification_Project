# BCR_AudioClassificationProject
Moving towards sustainability development, this project was carried out to prototype the Beverage Container Return(BCR) Scheme for plastic bottles or aluminium cans, launched in April 2026, on a smaller-scale to aid in recycling automation on the edge through a low-cost, low-power and low-latency solution. This report aims to walk the reader through the entire process from ideation to end product implementation including challenges faced.

A short demonstration (1 minute) of the smart BCR bin can be accessed through the youtube link or in the submission folder to aid in understanding the system workflow and overall functionality towards the implementation of the final product: https://youtu.be/jP9-McRUhSU

Hardware used for the project: Arduino Nano 33 Sense BLE, SG90 Servomotor, 4 x AA Battery Pack

Intended workflow: User finishes beverage -> Throw Beverage Container into BCR Bin -> IMU detects impact -> Audio Impact above amplitude threshold -> Model runs inference -> Servo sorts to correct bin
